# 🌾 Global Solution - AgroGestão Mobile

## 📖 Descrição da Solução
O **AgroGestão** é uma aplicação mobile (React Native/Expo) projetada para empoderar pequenos agricultores no gerenciamento eficiente de suas áreas de plantio. A solução permite que o produtor mantenha o controle de suas culturas, safras e tamanhos de terreno na palma da mão, através de uma interface intuitiva e fluida.

Este aplicativo consome dados diretamente de uma API RESTful baseada em arquitetura escalável (Java/.NET), garantindo segurança e confiabilidade nos dados armazenados em nuvem. A gestão digital diminui o desperdício, ajuda no planejamento da safra e impulsiona a produtividade no campo.

## 📋 Requisitos Avaliativos Cumpridos
* **Telas e Navegação:** Implementação de navegação usando React Navigation, contendo 5 telas (`Dashboard`, `PlantingsList`, `PlantingForm`, `PlantingDetails` e `Profile`).
* **Integração API (CRUD):** Uso da biblioteca `axios` com operações completas (POST, GET, PUT, DELETE) e tratamento de erros visuais (Loaders, Alerts e states customizados).
* **Arquitetura Clean e Componentização:** Projeto modularizado em `src`, com divisão clara de responsabilidades (Rotas, Services, Screens, Components).
* **UI/UX e Identidade Visual:** Interface coerente e moderna com cores voltadas ao agronegócio e respeito aos padrões de espaçamento do iOS/Android.

## 🚀 Como Executar o Projeto Localmente
1. Certifique-se de ter o Node.js instalado.
2. Clone este repositório.
3. No terminal, execute `npm install` para instalar as dependências.
4. Execute `npx expo start` para iniciar o servidor.
5. Escaneie o QR Code usando o aplicativo Expo Go no seu smartphone ou inicie o emulador configurado na sua máquina.
